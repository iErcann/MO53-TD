\doxysubsection{Detailed Description}
Classe abstraite représentant un chargeur de données. 

Cette classe abstraite définit l\textquotesingle{}interface commune pour tous les chargeurs de données. Un chargeur est responsable de charger des données à partir d\textquotesingle{}une source spécifiée dans un objet de type {\ttfamily \doxylink{classstorage__type}{storage\+\_\+type}}.
